ChaiCSS

A lightweight utility-first CSS engine built with JavaScript

How It Works

ChaiCSS scans the DOM, detects classes starting with chai-, parses them into style objects, and applies inline styles dynamically.

Why ChaiCSS?

Utility classes like chai-p-20 and chai-bg-red make styling fast and reusable.

Fast Styling

Apply styles instantly using class names without writing separate CSS files.
