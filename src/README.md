# ChaiCSS

A lightweight utility-first CSS engine built with vanilla JavaScript.

ChaiCSS allows you to write utility classes directly in your HTML (like `chai-p-20`, `chai-bg-red`) and automatically converts them into inline styles by scanning the DOM.

## Why ChaiCSS?

Writing custom CSS for every component can be repetitive and time-consuming. Utility-first frameworks solve this problem by letting you compose styles using small reusable classes.

ChaiCSS is a simplified implementation of that idea. It focuses on understanding how such systems work internally by building one from scratch using JavaScript.
